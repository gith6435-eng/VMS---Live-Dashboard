# VMS---Live-Dashboard

# Real-Time Business Metrics Dashboard for VMS Techs

## Project Description
This project is a **real-time interactive business dashboard prototype** developed for VMS Techs to support faster and more accurate decision-making.

The dashboard displays important business performance indicators such as sales, revenue, margin, renewal pipeline, and partnership index in a visually appealing format.

It automatically refreshes the data at regular intervals and updates charts dynamically, allowing business analysts to monitor trends and key metrics in real time.

The main objective of this project is to **reduce uncertainty caused by delayed reports** and provide quick insights for better strategic decisions.

---

## Problem Statement
Decision-makers at VMS Techs face significant risks due to delayed or inaccurate data insights, which can lead to costly business decisions.

This project aims to solve that problem by providing a **live dashboard** that visualizes key business metrics and trends in real time.

---

## Features
- Real-time automatic data refresh
- Manual refresh button
- KPI summary cards
- Bar chart for metric comparison
- Line chart for trend analysis
- Responsive and user-friendly interface
- Smooth chart updates using JavaScript
- Built using Chart.js

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

---

## Dashboard Metrics
The dashboard tracks the following business indicators:

- Enterprise Software Sales
- Cloud Services Revenue
- Managed Operations Margin
- Customer Renewal Pipeline
- Strategic Partnership Index

---

## How It Works
1. JavaScript fetches business metric data
2. The values are processed dynamically
3. Summary cards are updated
4. Charts refresh automatically every 5 seconds
5. Users can also refresh manually

---

## Purpose
The purpose of this project is to help business analysts and decision-makers at VMS Techs by providing **live insights and trend tracking**, enabling faster and smarter decisions.
